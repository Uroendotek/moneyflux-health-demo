Reglas financieras del caso

El flujo financiero depende de si el paciente tiene seguro y de si el caso termina como ambulatorio o hospitalario.

Paciente con seguro La póliza permite activar el flujo de cobertura y validación del deducible.
Si es ambulatorio El paciente paga un porcentaje del deducible conforme a la póliza.
Si se traslada a hospital El deducible se liquida en hospital y el expediente llega con estudios y diagnóstico.
Si no tiene seguro La plataforma puede marcar el caso para pago directo o ruta alternativa.
